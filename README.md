# BarrocIntens Applicatie

## 📌 Over de applicatie
De BarrocIntens applicatie is een C# applicatie die gebruik maakt van Entity Framework Core en een MySQL-database.

Met deze applicatie kunnen medewerkers van Barroc Intens werken met:
- Medewerkers
- Klanten
- Taken
- Koffiezetapparaten

De applicatie bevat verschillende gebruikersrollen zoals Eigenaar, Finance, Sales, Inkoop, Maintenance en Planner.

---

## ⚙️ Vereisten

Voor het gebruiken van deze applicatie heb je nodig:

- Windows
- Visual Studio 2022 of nieuwer
- .NET 6.0 of hoger
- MySQL Server 8.0+
- MySQL Workbench (optioneel)

Gebruikte technieken:
- C#
- Entity Framework Core
- MySQL
- BCrypt (voor wachtwoorden)
